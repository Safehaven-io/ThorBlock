    What is SafeID?¶

    SafeID is an identity provider built on the OAuth2 protocol that provides a secure method for log-on into several platforms with a single username and password. Essentially, it provides our community with a one-time registration of credentials to be used across all SafeHaven/SafeTech related sites/applications. We call this Single Sign-on or SSO. Translated: Whatever platform(s) you utilize, SafeID will only require one set of login credentials.

    SafeID provides:¶

    • Convenient and secure method of seamless login credentials
    • Ease-of-use
    • Security of the Safe Haven brand

    SafeID also will function as a hub for:

    • Identity management
    • Access management
    • Payment management (SafeTech product subscriptions)
